# Ejercicios de Lógica - Java

## Reto 1: Sistema de Descuentos
Programa que solicita la edad del cliente y el monto de compra, y aplica descuentos según las siguientes condiciones:
- Menor de 18 años → 5% de descuento
- Entre 18 y 60 años → 10% de descuento
- Mayor de 60 años → 20% de descuento

El programa muestra el descuento aplicado y el total a pagar.

**Archivo:** `src/Reto1_Descuentos.java`

---

## Reto 2: Cajero Automático
Programa que simula un cajero automático con el siguiente menú:
1. Consultar saldo
2. Depositar dinero
3. Retirar dinero
4. Salir

El sistema valida que el usuario tenga saldo suficiente antes de realizar un retiro.

**Archivo:** `src/Reto2_Cajero.java`

---

## Cómo ejecutar
```bash
cd src
javac Reto1_Descuentos.java && java Reto1_Descuentos
javac Reto2_Cajero.java && java Reto2_Cajero
```
